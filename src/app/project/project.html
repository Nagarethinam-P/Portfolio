Developing a Real-Time Data Analytics Framework using Hadoop

Duration:Jan 2025 ‚Äì Apr 2025


Project Overview

This project focuses on developing a real-time data analytics framework to monitor and filter unwanted messages in Online Social Networks (OSNs). The system combines big data technologies with machine learning to deliver fast, customizable, and scalable content moderation.

Technologies Used
  • Big Data: (HDFS, MapReduce, Hive)
  • Backend: PHP
  • Database: MySQL
  • Web Server:Apache
  • Machine Learning: Soft classifier (Supervised Learning, NLP)
  • Platform: Windows 7 (Client-Server Architecture)
Key features
  • Developed a rule-based filtering engine block unwanted posts on social network user walls.
  • Integrated a learning soft classifier to auto-label and filter content based on user preferences.
  • Designed a user-controlled content moderation system for enhanced privacy and security.
  • Built web dashboards for configuration and visualization of filtering results.
  • Demonstrated efficient real-time data processing using Hadoop components.
Future Enhancements:
  • Incorporate online learning models with user feedback.
  • Add trust-based access controlandauto-suggested filtering rules.
  • Improve usability with a more intuitive GUIfor managing filters and trust settings.
